# 3DEM-2019

3DEM Neuroglancer Workshop: Using Neuroglancer's python integration.

## Set up Jupyter in a new virtualenvironment

```
python3 -m venv ENV
source ENV/bin/activate
pip install -r requirements.txt
jupyter-lab
```
